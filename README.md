<h1>オリジナルキャラクターサイト</h1>
<br>
使用言語<br>
・HTML<br>
・css<br>
・Java Script<br>
<br>
オリジナルキャラクターの紹介をメインとしたサイトの制作をしました。<br>
コーディングでは、わからない部分をchatGPTに聞きながら自分の手で打ちこみ、<br>
そのまま書いて終わりではなく、理解することを意識して取り組みました。<br>
キャラクターが実際に出てくるページでは開くたびにランダムで出てくるイラストが変わるよう設計し、<br>
スマホ画面へのレスポンシブにも対応しています。<br>
