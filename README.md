<h1>オリジナルキャラクターサイト</h1>
<br>
使用言語<br>
・HTML<br>
・css<br>
・Java Script<br>
<br>
オリジナルキャラクターの紹介をメインとしたサイトの制作をしました。<br>
HTML/CSSを用いたコーディングを行い、スマートフォン画面へのレスポンシブ対応も行なっています。<br>
キャラクターが表示されるページでは、ページを開くたびにランダムで異なるイラストが表示されるよう、JavaScriptを用いて実装しました。<br>
<br>
コーディングでは、不明な点を部分を調べたり、chatGPTに聞きながら自分の手で打ちこみ、そのまま書いて終わりではなく、役割などの内容を理解することを意識して取り組みました。<br>
